# ML Model Training

This repository contains a **Machine Learning model training workflow** built using Python and Jupyter Notebook. It performs exploratory data analysis, preprocessing, model training, evaluation, and visualization on the **Iris dataset**.

## 📌 Project Overview

This project demonstrates how to:

- Load and explore a dataset (`Iris.csv`)
- Perform data preprocessing and visualize features
- Train a machine learning classifier
- Evaluate the model performance using accuracy and confusion matrix

The aim is to build a working ML model using widely used libraries such as `pandas`, `scikit-learn`, `seaborn`, and `matplotlib`. :contentReference[oaicite:0]{index=0}
